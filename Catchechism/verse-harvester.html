Bakes every Read-tab proof-text into app/verses.js · ESV via your Worker

Before you start run on the live site

This must be opened from your deployed origin (the GitHub Pages URL) — that is the only origin your Cloudflare Worker allows to read the ESV API. Opening it from a local file or the editor preview will fail CORS. Place this file at the repo root next to app/ and visit it on the live site.

  1. Commit verse-harvester.html and open it on your live site (e.g. …github.io/repo/verse-harvester.html).
  2. Click Fetch all verses and let it run (it skips any already captured, so you can stop/resume).
  3. When complete, click Download verses.js and commit it over the existing app/verses.js.
